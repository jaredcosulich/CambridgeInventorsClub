The Code Puzzle

We'll be drawing pictures using The Code Puzzle.

In pairs you'll write on and organize cards to create a program. Then we'll take a photo of your program so that the app can execute your code and draw your program.

You'll find a cheatsheet at your table that provides more instructions and some tips.

Feel free to draw what ever you want!
